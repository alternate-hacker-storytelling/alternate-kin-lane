--- layout: post title: 'Deltacloud API Review' url: 'http://apievangelist.com/2010/11/17/deltacloud-review/' image: '' --- Deltacloud provides aggregated access to multiple cloud computing platforms in a single web service. Deltacloud currently supports seven major cloud computing platforms including Rackspace and Amazon EC2.

The Deltacloud API delivers a RESTful API using HTTP Authentication. The web service provides access to hardware profiles, realms, images, and instances. It provides a Ruby client, but no other language kits.
